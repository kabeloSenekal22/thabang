Coding tips

Common errors highly impossible to spot
  1. *<=* pay attention to that not *=<*
  2. *;* semicolon after writing conditions
  3. *,* coma at the end after declaring multiple variables on the same data type
  4. *.equals()* For comparing string not *==*
